# Simple Skills

一个面向长期学习者的离线时间账本应用。  
Simple Skills 聚焦「技能投入时长记录」这一核心场景：你定义目标技能与每日投入计划，通过正计时/倒计时持续打卡，并沉淀日志与统计数据。

- **在线访问**：[https://me-lixin.github.io/](https://me-lixin.github.io/)
- **核心定位**：长时段学习记录，而非番茄钟任务管理
- **数据策略**：纯前端、无后端、数据默认只保存在用户本地浏览器

---

## 产品特性

- **技能管理**：创建、编辑、删除技能，配置总学习时长与每日投入时长
- **双计时模式**：支持正计时与倒计时切换，适配不同学习习惯
- **学习日志**：暂停或结束计时后写入日志，支持逐日回溯加载与导出
- **趋势统计**：提供最近 30 天与最近 12 个月统计视图，支持图表交互
- **离线可用**：首次访问后缓存静态资源，后续可在无网络环境继续使用
- **数据迁移**：支持技能与日志 JSON 导出/导入

## 使用说明

### 1) 新建技能

1. 点击右上角 `+` 按钮。
2. 填写技能名称。
3. 设置总时长（`200 - 1000` 小时）。
4. 设置日投入（`3 - 15` 小时）。
5. 点击保存后，技能卡片会出现在主区域。

### 2) 开始/暂停学习计时

1. 在技能卡片点击开始按钮进入计时蒙版。
2. 使用计时开关开始或暂停。
3. 可切换正计时/倒计时模式。
4. 键盘快捷键：
   - `Space`：开始/暂停
   - `Esc`：退出计时蒙版并保存状态

### 3) 重置今日计时

- 在暂停状态下可使用重置按钮，仅重置当日计时相关数据（累计总投入保留）。

### 4) 查看日志和统计

- **日志页**：默认展示当天日志，点击“加载历史数据”逐天向前回溯。
- **统计页**：默认展示 30 天趋势，可切换至 12 个月视图。

### 5) 导入导出

- 打开入口：
  - 键盘：`Ctrl + E`
  - 触摸屏：连点屏幕 3 次
- 关闭入口：
  - 键盘：`Esc`
  - 触摸屏：连点屏幕 2 次
- 导出会生成两个文件：`日志.json`、`技能.json`。
- 导入时请分别选择对应文件，导入完成后建议刷新页面。

## 可靠性与数据安全

Simple Skills 采用本地优先设计，在不引入后端的前提下尽量提升可靠性：

- 计时过程会持续同步到 IndexedDB，降低意外中断损失。
- 页面隐藏/恢复时会基于本地时间戳进行时间补偿。
- 页面关闭前会触发持久化，尽量保存最后状态。
- 支持导出数据用于浏览器迁移或手动备份。

> 重要：若手动清除浏览器站点数据（缓存、IndexedDB、本地存储等），本地数据会丢失。  
> 建议定期导出备份。

## 快速开始（本地运行）

### 方式一：直接部署为静态站点（推荐）

将项目发布到任意静态托管平台（GitHub Pages、Netlify、Vercel 静态模式等）即可使用。

### 方式二：本地 HTTP 服务启动

由于浏览器同源策略限制，不建议直接双击 `index.html` 运行。请通过本地服务器启动：

```bash
python -m http.server 8000
```

然后访问：`http://localhost:8000`

## 技术实现

- **前端基础**：HTML + CSS + JavaScript（原生）
- **本地数据**：IndexedDB（技能与日志）
- **本地缓存**：Service Worker + Cache API
- **图表绘制**：Canvas 2D
- **轻量状态**：localStorage（自增 id 等）

---

## 兼容性说明

- 推荐浏览器：最新版 Chrome / Edge（Chromium 内核）
- 导入功能依赖文件选择器 API，部分浏览器可能不支持
- 旧浏览器在图表与离线能力上可能有降级表现

---

## 产品边界

Simple Skills 当前不覆盖以下能力：

- 云同步 / 多端实时同步
- 账号体系 / 团队协作
- 番茄钟策略与复杂提醒系统
- 高级数据分析（标签、项目、维度钻取等）

它的目标是：**把学习投入时间准确、低干扰地记录下来**。

## FAQ

### Q1：为什么我换浏览器后数据不见了？

数据存储在浏览器本地，不会自动跨浏览器迁移。请在旧浏览器导出后，在新浏览器导入。

### Q2：为什么离线访问失败？

请先在线打开一次，让 Service Worker 完成缓存；再在离线环境访问。

### Q3：为什么导入失败？

请确认导入的是系统导出的标准 JSON 文件，且文件内容未被手动修改。

## 设计初衷

对高价值技能而言，长期投入比短期冲刺更关键。  
Simple Skills 的设计目标不是“花哨效率工具”，而是一个可长期依赖的学习时间账本。
